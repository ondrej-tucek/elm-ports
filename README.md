# Elm Ports

Example using of ports with Plotly.


### Install:
```
$ npm install
$ elm-package install -y
```

### Launching app:
```
$ npm start
```

* Access app at `http://localhost:8080/`
* Or live [demo](http://elm-ports.surge.sh/)
